# Backend Documentation

This folder documents backend architecture, module design, service patterns, and domain behavior.

## Current Backend Shape

- Express app factory.
- Feature modules under `backend/src/modules`.
- Prisma data access.
- Zod validation.
- JWT authentication.
- Synchronous score recomputation after assessment changes.
